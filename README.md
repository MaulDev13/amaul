This repository contains the source code for my personal website, built and hosted using GitHub. It serves as a space where I share a bit about myself, showcase my projects, and document what I’ve been working on. I’m passionate about game development, and so far most of my exploration has been through Unity, which is currently the main engine I use. That said, it’s not that I lack interest in other platforms like Godot, Roblox, or similar tools—I’m always open to learning and trying new things when the opportunity comes.

Currently, I work as a backend developer at a state-owned enterprise (BUMN), where I gain hands-on experience building and maintaining systems. I previously studied Information Technology (S1) at Universitas Riau (UNRI) starting in 2015, which laid the foundation for my journey in tech. In addition to building games and web projects, I also experiment with DevOps practices, including tools like Jenkins, Docker, and deployment automation. I occasionally contribute to or take on different roles in collaborative projects as well. This site will continue to evolve over time as I keep learning, building, and exploring new ideas.
