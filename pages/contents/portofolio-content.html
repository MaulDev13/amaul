Personal Portfolio System

A lightweight portfolio application built using HTMX and a Go backend API, deployed on GitHub Pages as a static-first frontend.

The system explores server-driven UI patterns, reducing reliance on heavy frontend frameworks while maintaining clean structure and efficient content loading.

Role: Fullstack Developer

Tech: HTMX, Go, HTML, CSS, GitHub Pages

Key Contributions

• Designed modular page structure using reusable HTML components

• Implemented dynamic content loading with HTMX

• Built simple routing system for partial page updates

• Integrated static frontend with Go-based API

• Deployed frontend via GitHub Pages for simplicity and accessibility
